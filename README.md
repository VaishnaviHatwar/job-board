# 🧑‍💼 Job Board — MERN Stack + AI

A full-stack job board web application built with the MERN stack where employers can post jobs and candidates can search and apply.

## 🌐 Live Demo
- **Frontend:** https://ubiquitous-platypus-360f5a.netlify.app
- **Backend:** https://job-board-kvp0.onrender.com

## 📁 GitHub Repository
https://github.com/VaishnaviHatwar/job-board

## ✨ Features
- JWT Authentication (Employer & Candidate)
- Employer Dashboard — Post & manage jobs
- Candidate Dashboard — View applications & status
- Job Search with filters
- Resume Upload (Multer)
- Email Notifications (Nodemailer)
- AI Job Assistant (Groq API — Llama 3.3)
- Mobile Responsive

## 🛠️ Tech Stack
**Frontend:** React.js, React Router, Axios, Tailwind  
**Backend:** Node.js, Express.js  
**Database:** MongoDB Atlas  
**AI:** Groq API (Llama 3.3 70B)  
**Hosting:** Netlify (Frontend) + Render (Backend)

## 🚀 Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
